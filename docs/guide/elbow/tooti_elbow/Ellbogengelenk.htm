Schmerzen im Ellbogengelenk

Schmerzen, welche vom Ellbogengelenk selbst verursacht werden, werden häufig nur in der rTiefe wahrgenommen und können nicht genau lokalisiert werden. Die Beschwerden werden meistens bei voller Streckung und Beugung (Endstellungen) wahrgenommen. Meistens ist eine vorausgehende Verletzung (Knochenbruch, Knorpelschaden, Bänderriss, Ausrenkung) die Ursache der Beschwerden. Über längere Zeit bestehende Schmerzen können auf eine Abnutzung des Gelenkes hindeuten (Arthrose). Zur Abklärung ist meist eine ärztliche Untersuchung, eine Röntgenuntersuchung und/oder eine Magnetresonanztomographie notwendig. Gemäss Diagnose kann entweder mittels konservativer Massnahmen (Physioterhapie, Beweglichkeitsübungen, Kräftigung) oder allenfalls mittels Gelenkspiegelung / Operation behandelt werden.
