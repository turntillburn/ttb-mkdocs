Hammerfortsatz (Hamulus ossis hamati)

Schmerzen im Bereich des Hammerfortsatzes können auf einen Bruch dieses Knochens zurückgeführt werden. Insbesondere bei Sportarten, welche einen Schläger benutzen (Golf, Tennis) kann es zu einem Ermüdungsbruch kommen. Andererseits kann ein starker Schlag gegen den Kleinfingerballen zu einem akuten Bruch führen. Meist kann dieser nur mit speziellen Röntgenbildern (Computertomographie) dargestellt werden. Falls eine Ruhigstellung mit Gips keinen Erfolg bringt muss das abgebrochene Knöchelchen operativ entfernt werden.
