Instabilität Daumensattelgelenk

Bei Schmerzen im Bereich des Daumensattelgelenkes mit Schwäche und Ausweichbewegungen (giving way) liegt meist eine Bandinstabilität des Gelenkes vor. Ursache dafür können eine vorausgehende Verletzung, eine anlagebedingte Bandlockerung, knöcherne Stellung mit flachen Gelenkflächen oder beginnende Abnützungserscheinungen sein. Mittels speziellen Röntgenaufnahmen kann eine Instabilität dargestellt werden. Um eine daraus resultierende vermehrte Gelenksabnützung zu verhindern kann eine Bandnaht oder Bandplastik durchgeführt werden. Im Falle einer fortgeschrittenen Arthrose in dem Gelenk müssen verschiedene operative Verfahren in Betracht gezogen werden.
