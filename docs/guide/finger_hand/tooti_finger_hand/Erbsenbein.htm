Erbsenbein (Os pisiforme)

Schmerzen im Bereich des Erbsenbeines können auf eine Gelenksentzündung, Schleimbeutelentzündung, einen Knochenbruch, eine Artrhose oder eine begleitende Sehnenscheidenentzündung des ellenseitigen Handgelenksbeugers (Flexor carpi ulnaris) zurückzuführen sein. Die Therapie richtet sich nach der Ursache, meist sind Ruhigstellung mit Schiene physiotherapeutische Massnahmen erfolgreich. Bei Knochenbruch oder fortgeschrittener Artrose kann mittels Operation das Knöchelchen entfernt werden.
